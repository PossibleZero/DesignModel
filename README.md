单例模式，列举了常用的单例方式

单例的好处：
1. 单例（这不是废话吗）
2. 延迟加载
3. 线程安全
4. 没有性能问题
5. 防止序列化产生新对象
6. 防止反射攻击

单例的弊端：

1、单例模式一般没有借口，扩展很困难，扩展需要改代码没有其他途径。
2、单例对象如果持有Context，很容易引起内存泄露，传递最好是ApplicationContext
